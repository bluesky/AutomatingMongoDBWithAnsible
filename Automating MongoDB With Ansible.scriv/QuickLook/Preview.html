  • MongoDB Collection

    • MongoDB Roles

      MongoDB Roles


      mongodb_repository - Add a MongoDB repository file to the host.

      mongodb_install - Installs MongoDB packages.

      mongodb_dba - Install various tools on the node that are useful to the MongoDB Administrator.

      mongodb_replicaset_cluster - Setup a simple 3 node MongoDB replicaset.

      mongodb_2shard_cluster - Setup a 6 node, 2 shard MongoDB cluster.

      mongodb_backup - Add a simple backup job to the node and copy it to s3.

      mongodb_nagios - Add Nagios checks to the MongoDB node.
